## Build setup

> **Note:** Replace `modules/config.js` with your twitter app tokens, learn more [here](https://developer.twitter.com/en/docs)

```bash
git clone https://github.com/AsaoluElijah/endsars-bot.git
cd endsars-bot
npm install
```
